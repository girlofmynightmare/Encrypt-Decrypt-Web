# Encrypt / Decrypt Web Tool

Simple web app for text encryption and decryption.
Input text, process, get output. Done.

# Features

- Encrypt text
- Decrypt text
- Runs fully in browser
- No external dependencies

# Usage 

- Open the website
- Enter text
- Choose encryot or decrypt
- Process and get result

# Tech Stack

- HTML
- CSS
- JavaScript

# Notes

This project is for learning and basic use.
Not intended for high-security or sensitive data.

# Security Notes

This project does not contain intentional backdoors, trackers, or hidden data exfiltration mechanisms.
All encryption and decryption processes run locally in the user's browser.
